### Chatbot 이란?

- 텍스트나 음성으로 인간과의 대화를 통해 동작하는 모든 프로그램
- 기원 : Turing Test
  - 기계가 인간과 구분할 수 없을 정도로 대화가 가능하다고 판단되면 기계가 지능이 있다고 판별하는 테스트
